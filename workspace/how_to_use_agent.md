# Як працювати з AI агентом: гайд для щоденного використання

> Зберігайте цей файл відкритим. Він відповідає на питання "як сформулювати запит" для будь-якої задачі.

---

## ГОЛОВНЕ ПРАВИЛО

Чим більше контексту ви даєте агенту — тим краща відповідь.
Агент — як новий співробітник: він розумний, але не знає вашого бізнесу без вступу.

**Формула будь-якого промпту:**
```
Роль + Контекст продукту + Конкретна задача + Формат відповіді
```

---

## КРОК 1. Завжди починайте сесію з цього рядка

```
Продукт: AIR Music Distribution
Роль: [стратег / SMM / таргетолог / копірайтер / аналітик / PM]
Контекст: [вставити з brief.md або коротко описати ситуацію]
Задача: ...
```

Це займає 30 секунд — але агент одразу "стає" потрібним спеціалістом.

## КРОК 2. Ролі та їх промпти

### Проджект Менеджер ← ПОЧИНАТИ КОЖЕН ТИЖДЕНЬ З ЦЬОГО
```
Ти — PM маркетингового відділу AIR Music Distribution.
У мене є 3 продукти, але зараз фокус на AIR Music.
Ціль: 10 контрактів/місяць.
Ресурс: я одна людина + AI агент.
Зараз є: [що зроблено за останній тиждень].
Дані за тиждень: Google ліди [X], Meta ліди [X], MQL [X].
Склади план на тиждень: що мені робити, що робить AI. Пріоритети по важливості.
```

### Маркетолог-стратег
```
Ти — старший маркетолог-стратег AIR Music Distribution.
Ось бриф: [вставити brief.md або ключові секції].
Задача: [опис].
Формат відповіді: гіпотеза → обґрунтування → план дій → метрики успіху → ризики.
```

### Аналітик
```
Ти — маркетинговий аналітик AIR Music Distribution.
Ось дані: [вставити — CTR, CPA, MQL, ліди тощо].
Що бачимо → чому це відбувається → що з цим робити → що відстежувати далі.
```

### Таргетолог (Meta)
```
Ти — таргетолог з 5+ роками досвіду в Meta Ads.
Продукт: AIR Music Distribution. ЦА: mid-tier Filipino artists, 1.5K–500K monthly listeners.
TOV: спокійний, людяний, pain-first. Заборонено: хайп, "best", "grow fast".
Задача: [запуск / аналіз / оптимізація].
```

### PPC (Google Ads)
```
Ти — PPC спеціаліст Google Ads.
Продукт: AIR Music Distribution. Ринок: Філіппіни.
Найкращий ad group зараз: royalty_issues (CTR 10.1%, CPA $3.4).
Задача: [семантика / аналіз / оптимізація].
```

### SMM менеджер
```
Ти — SMM менеджер AIR Music Distribution.
ЦА: Filipino independent artists, 1,500–500K monthly listeners, вже мають дистрибуцію але застрягли.
TOV: calm, clear, human, without hype. Pain-first завжди.
CTA у постах: "Comment CHECK below" (не "link in bio").
Мова: англійська з Taglish елементами де доречно.
Задача: [контент-план / пост / скрипт Reels].
```

### Копірайтер
```
Ти — копірайтер рівня Senior для AIR Music Distribution.
Аудиторія: mid-tier артисти, вже мають дистрибуцію, застрягли.
Мова: англійська. TOV: спокійний, людяний, pain-first. Без хайпу.
Заборонено: "best distribution", "upload everywhere", "grow fast".
Задача: [текст для реклами / лендінгу / поста].
```

### Лідогенератор / холодний аутріч
```
Ти — спеціаліст з холодного аутрічу для AIR Music Distribution.
ЦА: Filipino independent artists в Instagram з 2K–50K monthly Spotify listeners.
Вони вже мають дистриб'ютора але незадоволені сапортом.
TOV: людяний, не продажний. "Мене чують, не продають."
Мова: Taglish (hook) + англійська.
Задача: [DM скрипт / email / follow-up послідовність].
```

## КРОК 3. Щотижневий ритм (15–30 хв на день)

| День | Сесія | Що робите з агентом |
|------|-------|---------------------|
| **Пн** | Планування | Промпт PM → план тижня |
| **Щоранку (15 хв)** | Денний брифінг | Дати дані → отримати пріоритети дня |
| **Ср** | Аналіз | Вставити дані з GA4/Meta → висновки та дії |
| **Пт** | Звіт + контент | Підсумок тижня → матеріали на наступний |

**Щоранковий промпт (15 хв):**
```
Ти PM. Вчора: [що зробила].
Ліди за вчора: Google [X], Meta [X] за GA4.
MQL: [X].
Що пріоритетно сьогодні?
```

**Щопонеділковий промпт (30 хв):**
```
Ти PM. Ось результати минулого тижня: [дані].
Ось цілі на цей місяць: 10 контрактів.
Склади план на тиждень.
```

**Щоп'ятничний промпт (20 хв):**
```
Ти аналітик. Ось що зробили цього тижня: [список].
Ось метрики: [дані].
Зроби звіт: що спрацювало, що ні, що змінити наступного тижня.
```

## КРОК 4. Шаблони для частих задач

**Для стратегії:**
```
Я маркетолог AIR Music Distribution. Ось поточна ситуація: [дані].
Ось мета: [мета]. Запропонуй 3 стратегії з обґрунтуванням і пріоритетністю.
```

**Для аналізу даних:**
```
Проаналізуй ці дані: [вставити].
Що вони говорять про поведінку аудиторії?
Які 3 головні висновки і що робити далі?
```

**Для порівняння з конкурентами:**
```
Порівняй AIR Music Distribution з [конкурент] за: ЦА, УТП, канали, ціна.
Де наші слабкі місця і можливості?
```

**Для заголовків реклами:**
```
Ти копірайтер для Google Ads. Продукт: AIR Music Distribution. Ринок: Філіппіни.
ЦА: mid-tier артисти які сумніваються чи можна довіряти дистриб'ютору.
Напиши 5 варіантів headline (макс 30 символів кожен).
Контекст: [запит аудиторії, напр. "is distrokid safe"].
Тон: впевнений, людяний. Не "best", не "all-in-one".
```

**Для контент-плану Instagram:**
```
Ти SMM менеджер AIR Music Distribution.
ЦА: Filipino independent artists, 1.5K–500K listeners, вже мають дистрибуцію.
TOV: calm, clear, human, pain-first. CTA: "Comment CHECK below".
Склади контент-план на 2 тижні: 6 постів (3/тижд).
Теми з болів: "release вийшов нічого не відбулось", "support мовчить", "royalties незрозумілі".
Формат: тема / перший рядок (hook) / тип (Reels/карусель/пост) / CTA.
```

**Для холодного DM:**
```
Ти спеціаліст з холодного аутрічу AIR Music Distribution.
ЦА: Filipino independent artists в Instagram, 2K–50K Spotify listeners.
Вже мають дистриб'ютора але незадоволені. TOV: людяний, не продажний.
Мова: Taglish hook + англійська.
Напиши 3 варіанти першого DM (макс 3 речення).
+ Follow-up: що писати якщо не відповіли через 2 дні.
```

## КРОК 5. Кризові тригери — коли зупинятись

| Ситуація | Дія |
|----------|-----|
| MQL Rate < 40% | Зупинити масштабування. Запитати агента: "чому якість лідів низька?" |
| CPA > $380 (Ф1) | Призупинити 30% слабких адсетів. Переглянути ICP. |
| Meta дає початківців | Не збільшувати бюджет. Фікс таргетингу першочерговий. |
| Start Rate Typebot < 25% | Прибрати текст перед кнопкою. |

## КРОК 6. Структура знань у репозиторії

```
workspace/
  products/
    air_music_distribution/
      brief.md          ← ГОЛОВНИЙ ДОКУМЕНТ (21 секція)
    product_2/brief.md
    product_3/brief.md
  roles/
    roles_prompts.md    ← Всі 17 ролей з промптами
  weekly/
    week_current.md     ← Поточний тижневий план
    week_01.md
  how_to_use_agent.md   ← ЦЕЙ ФАЙЛ
```

**Як вставляти бриф у промпт:** відкрийте `brief.md`, скопіюйте потрібні секції (1-6 для більшості задач, секції 19-20 для реклами).

---

## ЩО НЕ ТРЕБА РОБИТИ

- Не питати без контексту ("напиши пост для Instagram" — занадто загально)
- Не починати нову задачу без назви продукту і ролі
- Не масштабувати рекламу якщо MQL < 40%
- Не приймати рішення по Meta dashboard — тільки GA4
