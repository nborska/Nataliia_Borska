# Поточний тижневий план — Фаза 1 (Квітень–Червень 2026)
**Контекст:** M1 завершено. Головна проблема — якість лідів (Meta дає початківців, не mid-tier). Є 3 критичні технічні проблеми які треба вирішити першими.

---

## 🔴 КРИТИЧНО ЗАРАЗ (це робити цього тижня)

### 1. Виправити Attribution Meta (30 хв, Наталія)
**Проблема:** Meta рахує 70 конверсій, GA4 показує 10. Різниця 7x — приймаємо хибні рішення.
**Дія:** У Meta Ads Manager → Settings → Attribution → змінити на **"1-day click only"**
**Після:** Всі рішення по Meta — тільки за GA4, не за Meta dashboard

### 2. Виправити Typebot (1 год, розробник або Наталія)
**Проблема:** 83% людей не натискають Start бота через вступний текст
**Дія — Artists Check бот:** Прибрати весь текст перед першою кнопкою. Одразу кнопка.
**Дія — Personal Manager бот:** Скоротити з 10+ кроків до 4. Не три поля підряд.
**Очікуваний результат:** Start Rate з 17% → 35–45%, завершень 50–70 замість 14

### 3. Мінус-слова Google (30 хв, Андрій)
**Проблема:** howto_keywords дають неякісний трафік (CPA $18 замість $6)
**Додати мінус-слова:** free, download, karaoke, lyrics, tutorial, beginner, "how to", "what is", diy
**Кампанія:** Personal Manager — howto_keywords ad group

## 🟡 ВАЖЛИВО ЦЕЙ ТИЖДЕНЬ

### 4. Масштабувати royalty_issues (30 хв, Андрій)
- CTR 10.1%, конверсія 71.4%, CPA ~$3.4 — це золота група
- Збільшити ставку до 12 AED
- Розширити ключові слова: spotify royalty calculator, music streaming payout, artist earnings from spotify
- Після підтвердження CPA < $6 → збільшити бюджет до 60–80 AED/день

### 5. CTA на лендінгу Manager вище (розробник)
- Конверсія: Check 38.9% vs Manager 15.8%
- Підняти кнопку "Apply" вище на сторінці (до першого скролу)

### 6. Запустити Ретаргетинг (після 6–8 тижнів роботи кампаній)
Три аудиторії:
- "Відвідав лендинг + ввів Spotify URL" — найгарячіша
- "Почав Typebot але не завершив"
- "Переглянув 75%+ кейс-відео"

## 📊 МЕТРИКИ ДЛЯ МОНІТОРИНГУ (щотижня)

| Метрика | Норма | Де дивитись |
|---------|-------|-------------|
| CTR оголошень Google | > 2% | Google Ads |
| CTR Meta | > 1% | GA4 (не Meta!) |
| Conversion Rate Check лендінг | > 30% | GA4 |
| Conversion Rate Manager лендінг | > 15% | GA4 |
| Start Rate Typebot | > 25% | Typebot Analytics |
| CPA Check | < $8 | GA4 |
| CPA Manager | < $12 | GA4 |
| MQL Rate | > 40% | HubSpot |
| Нові ліди/тиждень | > 5 | HubSpot |
| Частота Meta | 1.5–2.5 | Meta Ads |

**Правило:** Якщо MQL Rate < 40% — зупиняємось, не масштабуємо, розбираємось в причині.
