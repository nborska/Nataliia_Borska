## ПРОДУКТ: "Продюсер у кишені"

### Головний слоган
> **"Курс дає знання. Бот дає готовий пост."**

### Маніфест (основа всього маркетингу)
```
Не треба аналізувати конкурентів.
Не треба проводити інтерв'ю з ЦА.
Не треба робити аналіз ринку.
Не треба розбиратися у маркетингу.
Не треба наймати продюсера.
Не треба іти на ще один курс.
Не треба мучитися бо нема ідей.
Не треба копіювати інших.

Бот все зробить за тебе.
Тобі потрібно лише дати орієнтири.
Це не забере весь твій день.

Блог у легкості. Продюсер у тебе в кишені.
```

### УТП (одне речення)
> "Знання маркетолога і продюсера вже зашиті всередині. Ти просто даєш орієнтири — і він працює індивідуально під тебе. Завжди на зв'язку."

### Чому це сильно — по кожному пункту
```
"Знання зашиті"           → не треба вчитись, експертиза вже всередині
"Просто даєш орієнтири"   → мінімум зусиль, не складні форми
"Індивідуально під тебе"  → не шаблон для всіх, твоя ніша і стиль
"Завжди на зв'язку"       → не курс який подивилась і забула, 24/7
"Не забере весь день"      → 30 хвилин і готово
```

### Позиціонування vs конкуренти
```
Курс за $500:          знання є → застосування на тобі → більшість не робить
Продюсер за $500/міс:  робить → але дорого і не завжди твій
ChatGPT:               допомагає → але не знаєш що питати, немає системи
Продюсер у кишені:     знання + дія + індивідуально + доступно + 24/7
```

- Мова: Українська

**Що робить бот:**
```
1. Клієнт вводить код доступу після оплати
2. Онбординг: 8 питань (ніша, ЦА, цілі, стиль, страхи)
3. Генерує пакет:

4. Щотижня: /content → нова порція ідей
```

- Формат: мобільний додаток
- Функції: стратегія + бренд + контент-план + аналітика +
  планування + монтаж відео + каруселі + сторіс
- Ціна: $20-50/місяць підписка
- Потенціал: 1000 клієнтів = $29,000-49,000/міс
- Розробник: хлопець (повертається з Балі через 3 тижні)
- Статус: потрібно скласти ТЗ поки він у відпустці
